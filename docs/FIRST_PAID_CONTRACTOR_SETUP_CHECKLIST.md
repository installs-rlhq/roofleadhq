# First Paid Contractor Setup Checklist

Date: 2026-06-05

## Purpose

Use this checklist to confirm the first paid contractor is ready for the RoofLeadHQ Founder-Led Launch Program.

This is the final setup handoff between intake, kickoff email, and daily operations. It keeps the launch framed around review, monitoring, and helping the contractor book inspections / book appointments without overpromising automation or outcomes.
